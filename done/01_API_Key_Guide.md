# API 키 발급 및 연동 가이드

본 문서에서는 주식 리포트 자동화 시스템 구동에 필요한 3가지 핵심 서비스(Google AI Studio, Notion, Gmail)의 API 키 및 설정 방법을 안내합니다.

---

## 1. Google AI Studio (Gemini) API Key 발급

무료로 텍스트 요약을 생성하기 위해 Gemini 모델을 사용합니다.

1. **접속**: [Google AI Studio](https://aistudio.google.com/) 에 구글 계정으로 로그인합니다.
2. **API 키 생성 메뉴**: 좌측 메뉴바에서 `Get API key` (또는 `API 키 얻기`) 메뉴를 클릭합니다.
3. **새 키 발급**: `Create API key` 버튼을 누르고, 새로운 프로젝트에서 생성하기를 선택합니다.
4. **키 복사 및 보관**: 생성된 키(영문과 숫자가 혼합된 긴 문자열)를 복사하여 안전한 곳에 메모합니다. (이후 `.env` 파일의 `GEMINI_API_KEY` 환경변수로 사용됩니다.)

---

## 2. Notion 워크스페이스 및 통합(Integration) 토큰 발급

사용자 리스트 데이터베이스 조회 및 설정을 로드하기 위해 사용합니다.

### 2.1. 통합(Integration) 생성

1. **접속**: [Notion Developers 로내 통합 페이지](https://www.notion.so/my-integrations) 에 로그인합니다.
2. **새 통합 만들기**: 좌측의 `+ 새 API 통합 만들기` 버튼을 클릭합니다.
3. **정보 입력**:
   - 이름: `Stock Report Bot` (예시)
   - 연결할 워크스페이스 선택
   - 기능: 콘텐츠 읽기, 콘텐츠 업데이트 (두 가지 권한 필수 체크)
4. **저장 및 토큰 복사**: 하단의 `제출` 버튼을 누르면 **"프라이빗 API 통합 토큰"** (`secret_...` 형태)이 표시됩니다. 이를 복사해 보관합니다. (`NOTION_TOKEN` 환경변수로 사용)

### 2.2. 데이터베이스 연결 (공유)

1. 노션 워크스페이스에 **'사용자 리스트' 표(Database - 전체 페이지)**를 하나 만듭니다. (컬럼 예시: 이름, 이메일, 수신여부 등)
2. 해당 표가 있는 페이지 우측 상단의 `...` 아이콘을 클릭합니다.
3. 하단의 `연결` > `연결 추가`를 클릭하고, 방금 만든 `Stock Report Bot` (통합 이름)을 검색하여 추가(초대)합니다.
4. 웹 브라우저 주소창에서 해당 표의 URL을 보면 `https://www.notion.so/워크스페이스이름/abcdefg...?v=...` 형태입니다. 여기서 `abcdefg...` 부분이 **Database ID** 입니다. 이를 복사하여 보관합니다. (`NOTION_DATABASE_ID` 환경변수)

## 3. Gmail 앱 비밀번호 발급 (이메일 발송용)

Python 스크립트가 사용자의 Gmail 계정을 통해 메일을 보낼 수 있도록 권한을 설정합니다.

1. **2단계 인증 확인**: 이메일을 보낼 구글 계정으로 로그인 후 [Google 계정 보안 설정](https://myaccount.google.com/security) 으로 이동합니다. `Google에 로그인` 항목에서 `2단계 인증`이 켜져 있어야 합니다. (안 켜져 있다면 활성화 필요)
2. **앱 비밀번호 생성**:
   - 보안 페이지 상단 검색창에 **"앱 비밀번호"**(App passwords)를 검색해 들어갑니다. (또는 2단계 인증 메뉴 안쪽 하단에 위치)
3. **생성 옵션 선택**:
   - 앱 선택: `메일`
   - 기기 선택: `Mac` 또는 `기타(사용자 지정 이름: Python Bot)`
4. **생성 및 복사**: `생성` 버튼을 누르면 화면에 기기용 **16자리 영문 앱 비밀번호**가 나타납니다. 띄어쓰기 없이 복사하여 보관합니다. (기존 구글 계정 비밀번호 대신 사용됩니다. `GMAIL_APP_PASSWORD` 환경변수)
