# 주식 리포트 자동화 시스템 아키텍처 및 무료 AI 요약 방안 (Draft 2)

## 1. 시스템 핵심 구성

- **언어**: Python 3.10+
- **수집 (Crawler)**: `requests`, `BeautifulSoup4` (정적), `Playwright` (동적)
- **스케줄러 (Scheduler)**: GitHub Actions (3시간 주기 `cron` 트리거, 완전 무료)
- **데이터베이스 (DB)**: Notion Database API (사용자 리포트 구독/관심종목 테이블, 완전 무료 UI/DB)
- **요약 엔진 (AI)**: Gemini API (구독 중인 Gemini Advanced 권한 또는 Google AI Studio 무료 티어 활용)

## 2. 무료 AI 요약 시스템 구현 방안 (Gemini 활용)

현재 사용 중이신 **Gemini (Google)** 를 활용하면 추가 비용 없이 요약 시스템을 완벽하게 구축할 수 있습니다.

### 2.1. Google AI Studio (Gemini API) 무료 티어 이용

- **방법**: Google AI Studio(https://aistudio.google.com/)에 접속하여 API Key를 무료로 발급받습니다.
- **무료 제공량**: 현재 Gemini 1.5 Flash 모델의 경우 **1분에 15회 요청(RPM), 하루 1,500회 요청(RPD)** 까지 무료로 제공됩니다.
- **적합성**: 3시간마다 한 번씩(하루 8회) 수집된 기사와 커뮤니티 글을 묶어서 요약 요청을 보내는 방식이므로, 무료 한도 내에서 **매우 여유롭게 커버** 가능합니다.
- **장점**: 매우 빠르고 강력한 요약 능력을 갖추었으며, Python 패키지(`google-generativeai`)로 쉽게 연동 가능합니다.

### 2.2. 프롬프트 엔지니어링 (요약 퀄리티 높이기)

AI에게 원시(Raw) 데이터를 던질 때, 출력 형식을 강제하여 깔끔한 리포트를 얻어냅니다.

- **입력 데이터**: 크롤링한 뉴스 헤드라인 30개 + 종토방 인기글 제목/본문 요약 50개 등
- **요청 프롬프트 예시**:
  > "너는 전문 주식 애널리스트야. 다음 주식 뉴스 제목과 커뮤니티 반응 데이터를 바탕으로, 오늘 시장의 핵심 키워드 3가지를 뽑고 각각에 대해 3줄 이내로 매우 직관적이고 친절하게 요약해줘. (마크다운 형식 출력)"

## 3. 리포트 생성 및 발송 파이프라인

1.  **[수집 단계]**: 3시간마다 GitHub Actions 실행 -> 네이버 뉴스, 구글 트렌드, 디시 주식갤러리 등 지정된 사이트/API에서 데이터 수집 -> 파이썬 `pandas` 등을 사용해 노이즈 텍스트 정제.
2.  **[분석 단계]**: 정제된 텍스트 청크를 묶어 Gemini API 모델(`gemini-1.5-flash` 추천)에 전송 -> 핵심 이슈, 긍정/부정 여론 파악 및 요약 리포트 마크다운(또는 HTML 원본) 반환.
3.  **[조회 단계]**: 파이썬 코드에서 Notion API를 호출해 "리포트 수신 동의 상태(True)"인 사용자 리스트(이메일, 카카오톡 ID 등)를 가져옵니다.
4.  **[발송 단계]**:
    - **이메일**: 파이썬 빌트인 `smtplib` + Gmail 앱 비밀번호를 사용하여 수신자들에게 HTML 형태의 예쁜 리포트 메일 무료 발송.
    - **카카오톡**: 카카오 디벨로퍼스 REST API를 통해 사용자들에게 요약 텍스트 전송 (나에게 보내기 또는 친구 알림톡 구조).

## 4. 고려 사항 (크롤링 한계)

- **난이도 상**: '토스 주식 커뮤니티', '블라인드'는 모바일 앱/인증 기반이거나 강력한 봇 방어가 있어 자동화 수집이 막히거나 매우 어려울 수 있습니다.
- **대안**: 초기 버전을 론칭할 때는 수집이 쉬운 **네이버 뉴스, 네이버 종토방, 디시인사이드 식갤, 구글 트렌드** 4곳만 우선 타겟팅하여 시스템의 뼈대를 완성하고 발송 테스트를 거치는 것을 강력히 권장합니다.
